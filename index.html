Kho Kỷ Niệm

Private & Simple

Lưu trữ ảnh, kỷ niệm và nghe bài "Kho Báu (Trọng Hiếu)" — thay đường dẫn ảnh/âm nhạc khi bạn có.

Hướng dẫn nhanh:
  • Thêm link ảnh: mở file này, thay giá trị src của #img1, #img2, #img3 bằng link ảnh (URL trực tiếp tới ảnh .jpg/.png).
  • Thêm nhạc: thay <source src=""> bằng URL tới file mp3 của bạn (ví dụ file host trên GitHub, S3, hoặc dịch vụ lưu trữ khác). Nếu bạn muốn nhúng YouTube, thay phần audio bằng iframe YouTube (mình có thể giúp nếu cần).
Gợi ý: bạn có thể upload ảnh & mp3 lên repo GitHub riêng rồi lấy raw link để dán vào đây — dễ deploy lên GitHub Pages.
